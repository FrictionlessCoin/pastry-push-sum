public class PastryPushSum
extends Object
implements Application, ScribeMultiClient

An implementation of the Push-Sum protocol as a Pastry application.

This class implements the Application interface to send messages to random nodes in the ring and the ScribeMultiClient interface to broadcast notifications to all nodes.

Constructs and registers a new PastryPushSum application.

Parameters:
node - the node at which this application will be registered.
stepSize - the time between sending messages to the node to signal the start of the next step.
updateInterval - the number of steps between updating node value. If set to 0 the node values will never be updated.
valueReader - the ValueReader instance from which the node obtains its true value.
min - the domain-specific minimum possible value, used as a lower bound for the estimates.
max - the domain-specific maximum possible value, used as an upper bound for the estimates.
trace - if set to true, the node will print a notice about all sent and received messages to the standard output.

getEstimate

public double getEstimate()
Returns the average value of all nodes in the ring, as estimated by the Push-Sum protocol. The estimate is obtained by value / weight, bounded by the min and max values given in the constructor.

If the true value has been updated recently, this will return the true value instead, until one message from another node is received, as in this case it serves as a better estimate.

broadcastReset

public void broadcastReset()
Sends a reset notification to all nodes.

Since this is not a barrier, the resets introduce an error in the accuracy of the estimates. Overall it may be better not to do any resets, letting the contribution of stopped nodes "fade out".

deliver

public void deliver(Id id,
                    Message message)
Called when this node receives a message.

When receiving a timer message, sum up the values and weights from all messages received since the last step and send half of the new value and weight to itself and to a random neighbor.

When receiving a message from another node, store the received value and weight in a buffer for the next step.
